# ⚡ Самый быстрый способ (2 минуты) - ngrok

Если нужно ПРЯМО СЕЙЧАС протестировать с другом, используй ngrok!

## Что это?

ngrok создает туннель из интернета к твоему локальному серверу.

⚠️ **Минусы:**
- Работает только пока включен компьютер
- Бесплатная версия: URL меняется при каждом запуске
- Не для продакшена, только для тестов

✅ **Плюсы:**
- Настраивается за 2 минуты
- Не нужен GitHub
- Работает сразу

---

## Шаги:

### 1. Скачай ngrok

Перейди на https://ngrok.com/download и скачай для Windows

### 2. Зарегистрируйся

1. Создай аккаунт на https://dashboard.ngrok.com/signup
2. Скопируй свой authtoken

### 3. Настрой ngrok

Открой терминал и выполни:

```bash
ngrok config add-authtoken ТВОЙ_ТОКЕН
```

### 4. Запусти сервер

```bash
cd Huddle/HuddleServer
dotnet run
```

Сервер запустится на http://localhost:5000

### 5. Запусти ngrok (в новом терминале)

```bash
ngrok http 5000
```

Ты увидишь что-то типа:

```
Forwarding  https://abc123.ngrok.io -> http://localhost:5000
```

### 6. Обнови клиент

Открой `Huddle/Services/ChatService.cs` и измени:

```csharp
private readonly string _serverUrl = "https://abc123.ngrok.io/chathub";
```

Используй URL из ngrok (с https://)!

### 7. Готово!

Теперь можешь дать этот URL другу, и он сможет подключиться к твоему серверу!

---

## 🔄 При следующем запуске

1. Запусти сервер: `dotnet run`
2. Запусти ngrok: `ngrok http 5000`
3. Обнови URL в клиенте (он изменится)
