## 🎉 После деплоя

Не забудь обновить URL в клиенте!

Открой `Huddle/Services/ChatService.cs` и измени:

```csharp
private readonly string _serverUrl = "https://твой-сервер.railway.app/chathub";
```

Затем пересобери клиент:

```bash
cd Huddle
dotnet build Huddle.csproj
```

Готово! Теперь твой мессенджер работает через интернет! 🚀
