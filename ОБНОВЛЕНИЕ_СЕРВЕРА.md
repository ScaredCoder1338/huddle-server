# 🔄 Обновление сервера на Railway

## Что изменилось:

✅ Добавлена таблица Users в базу данных
✅ Регистрация пользователей через сервер
✅ Поиск пользователей через сервер
✅ Теперь пользователи с разных компьютеров могут найти друг друга!

## Как обновить сервер:

### Шаг 1: Загрузи изменения на GitHub

```bash
cd Huddle/HuddleServer
git add .
git commit -m "Add user registration and search"
git push
```

### Шаг 2: Railway автоматически задеплоит

Railway автоматически обнаружит изменения и задеплоит новую версию!

1. Перейди на https://railway.app
2. Открой свой проект huddle-server
3. Перейди в "Deployments"
4. Жди пока статус станет "Success" (2-3 минуты)

### Шаг 3: Проверь логи

В Railway перейди в "Deployments" → "View Logs"

Должно быть:
```
🚀 Huddle Server запущен на http://0.0.0.0:XXXX
📡 SignalR Hub: http://0.0.0.0:XXXX/chathub
```

### Шаг 4: Пересобери клиент

```bash
cd Huddle
dotnet build Huddle.csproj
```

### Шаг 5: Готово!

Теперь:
- Регистрация создает пользователей на сервере
- Поиск ищет пользователей на сервере
- Пользователи с разных компьютеров могут найти друг друга!

## 🎯 Тестирование:

1. Ты регистрируешься на своем компьютере
2. Друг регистрируется на своем компьютере
3. Ты ищешь его username - находишь!
4. Он ищет твой username - находит!
5. Общаетесь через интернет! 🚀

## ⚠️ Важно:

После обновления сервера база данных пересоздастся!

Нужно зарегистрироваться заново.
